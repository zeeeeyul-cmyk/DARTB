# 1️⃣ 개념 정리 

## 01. 데이터 분석이란

데이터 분석은 데이터를 이용해 어떤 문제에 대한 답을 찾는 과정이다.
단순히 숫자를 보는 것이 아니라, 필요한 데이터를 모으고 정리한 뒤 그 안에서 의미를 발견해 결론을 내리는 활동이라고 볼 수 있다.

또한 데이터 분석은 좁은 의미와 넓은 의미로 나누어 생각할 수 있다.
좁은 의미에서는 데이터를 직접 살펴보고 해석하는 과정, 즉 기초통계나 탐색적 데이터 분석 같은 분석 활동 자체를 뜻한다. 반면 넓은 의미에서는 데이터 수집, 정제, 처리, 분석, 모델링까지 포함한 전체 흐름을 의미한다. 그래서 데이터 분석은 단순 계산이 아니라, 문제를 해결하기 위해 데이터를 다루는 전반적인 과정이라고 정리할 수 있다.

## 02. 구글 코랩과 주피터 노트북

구글 코랩과 주피터 노트북은 파이썬 코드를 실행하고, 그 결과를 바로 확인하면서 학습할 수 있는 도구이다.
이 도구들의 가장 큰 장점은 코드, 실행 결과, 설명을 한 공간에서 함께 정리할 수 있다는 점이다. 그래서 데이터 분석을 처음 배우는 사람도 과정을 따라가며 이해하기 쉽다. 특히 이 책과 관련 자료는 코랩에서 실습을 진행할 수 있도록 구성되어 있어, 별도의 복잡한 설치 없이 브라우저 환경에서 분석을 시작할 수 있다.
주피터 노트북은 보통 개인 컴퓨터 환경에서 직접 실행하는 방식이고, 구글 코랩은 웹에서 바로 사용할 수 있어 접근성이 높다.
따라서 데이터 분석을 배우는 입장에서는 단순히 이론만 익히는 것이 아니라, 이런 실행 환경을 활용해 직접 데이터를 불러오고 다뤄 보는 경험이 중요하다는 점을 알 수 있다.

## 03. 이 도서가 얼마나 인기가 좋을까요?

이 강의에서는 “이 도서가 얼마나 인기가 좋을까요?”라는 질문을 통해, 데이터 분석이 실제 문제 해결에 어떻게 사용되는지를 보여준다.  

이 질문에 답하려면 단순한 느낌이나 추측으로 판단하면 안 되고, 도서와 관련된 실제 데이터를 확인해야 한다. 예를 들어 판매량, 대출 횟수, 평점, 리뷰 수 같은 자료를 모아 비교하면, 그 책의 인기를 좀 더 객관적으로 파악할 수 있다. 즉, 먼저 해결해야 할 질문을 정하고, 그 질문에 맞는 데이터를 찾은 뒤, 이를 분석 가능한 형태로 정리해 해석하는 과정이 필요하다. 이런 점에서 이 예시는 데이터 분석이 단순한 계산 기술이 아니라, 현실의 문제에 답을 주는 과정이라는 것을 잘 보여준다.

# 3️⃣ 확인 문제

## 문제 1.

> **🧚Q. 데이터 분석의 개념을 넓은 의미와 좁은 의미로 나누어 설명하고, 두 개념의 차이점도 함께 정리해주세요.**


```
데이터 분석은 넓은 의미와 좁은 의미로 나누어 볼 수 있다.
먼저 넓은 의미의 데이터 분석은 데이터를 다루는 전반적인 과정을 모두 포함하는 개념이다. 즉, 단순히 데이터를 해*하는 것만이 아니라 데이터 수집, 데이터 처리, 데이터 정제, 모델링, 그리고 결과 해석까지 전체 흐름을 포괄한다. 그래서 넓은 의미의 데이터 분석은 데이터로부터 의미 있는 정보를 얻기 위한 전체 과정이라고 볼 수 있다.
반면 좁은 의미의 데이터 분석은 그중에서도 실제로 데이터를 살펴보고 의미를 도출하는 분석 활동 자체에 초점을 둔다. 예를 들어 기초통계, 탐색적 데이터 분석, 가설 검정 등을 통해 데이터의 특징을 파악하고, 문제에 대한 해*이나 결론을 이끌어내는 과정이 여기에 해당한다.
두 개념의 차이점은 범위의 차이에 있다.
넓은 의미의 데이터 분석은 분석 이전의 준비 단계와 이후의 활용 단계까지 포함하는 전체 과정 중심의 개념이고, 좁은 의미의 데이터 분석은 그 안에서 실제로 데이터를 이해하고 해석하는 핵심 분석 활동 중심의 개념**이다.
즉, 정리하면 좁은 의미의 데이터 분석은 넓은 의미의 데이터 분석 안에 포함되는 개념이라고 할 수 있다.
```
